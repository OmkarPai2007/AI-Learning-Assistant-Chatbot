# AI Learning Assistant Chatbot

## Project Overview

An AI-powered educational chatbot developed using Botpress Studio. The chatbot helps users learn about Artificial Intelligence, Machine Learning, Deep Learning, Natural Language Processing (NLP), Generative AI, Prompt Engineering, Chatbot Development, and AI Careers.

## Features

* AI Learning Assistance
* Knowledge Base Integration
* Interactive Question Answering
* Domain Restriction to AI Topics
* Automated Fallback Response
* Educational Content Delivery

## SDG Alignment

SDG 4 – Quality Education

## Technologies Used

* Botpress Studio
* Autonomous Node
* Knowledge Base
* Rich Text Knowledge Source
* Webchat Emulator
